- `README.md` - သင်ခန်းစာအလိုက် Logic အကျဉ်းချုပ် မှတ်စုများ

## 📚 သင်ခန်းစာမှတ်စုများ

### 🔹 သင်ခန်းစာ (၁) - Perceptron Basics

- **ဖိုင်လမ်းကြောင်း:** `01_perceptron_basics.ipynb`
- **Logic အနှစ်ချုပ်:** \* AI ရဲ့ အခြေခံအကျဆုံး Neuron (အာရုံကြော) တစ်ခု ဘယ်လိုအလုပ်လုပ်လဲဆိုတာ လေ့လာခဲ့ပါတယ်။
  - Input တွေကို Weights တွေနဲ့မြှောက်၊ Bias ပေါင်းပြီး ရလာတဲ့ရလဒ်ကို Activation Function ထဲ ဖြတ်ရပါတယ်။
  - AI မော်ဒယ်တစ်ခု ပိုတော်လာတယ်ဆိုတာ နောက်ကွယ်က **Weights ($w$) နဲ့ Bias ($b$)** တန်ဖိုးတွေကို အဖြေမှန်ရအောင် စက်က အလိုအလျောက် လိုက်ညှိနေခြင်း ဖြစ်ပါတယ်။

---

### 🔹 သင်ခန်းစာ (၂) - Multi-Layer Neural Network အခြေခံ

- **ဖိုင်လမ်းကြောင်း:** `notebooks/02_neural_networks.ipynb`
- **Logic အနှစ်ချုပ်:** \* Perceptron တွေကို အလွှာလိုက် စုစည်းပြီး **Input -> Hidden -> Output** ဆိုပြီး ချိတ်ဆက်ပုံကို လေ့လာခဲ့ပါတယ်။
  - Matrix တွက်ချက်မှုတွေအတွက် **NumPy** ရဲ့ `np.dot()` ကို သုံးရပါတယ်။
  - Deep Learning မှာ အသုံးအများဆုံး **ReLU Activation Function** ရဲ့ အလုပ်လုပ်ပုံ (မိုင်နပ်စ်တန်ဖိုးတွေကို သုညဖြစ်စေပြီး ပလပ်စ်တန်ဖိုးတွေကို အတိုင်းပေးထွက်ပုံ) ကို လက်တွေ့ကုဒ်ရေးပြီး စမ်းသပ်ခဲ့ပါတယ်။

---

### 🔹 သင်ခန်းစာ (၃) - Loss Function နှင့် Gradient Descent အခြေခံ

- **ဖိုင်လမ်းကြောင်း:** `notebooks/03_loss_and_backprop.ipynb`
- **Logic အနှစ်ချုပ်:** \* AI ရဲ့ အဖြေနဲ့ အဖြေမှန် ကြားက ကွာဟချက်ကို **Mean Squared Error (MSE)** လိုမျိုး Loss Function တွေနဲ့ တိုင်းတာပုံကို လေ့လာခဲ့ပါတယ်။
  - အမှားကို နည်းနိုင်သမျှနည်းအောင် လုပ်ပေးတဲ့ **Gradient Descent** စနစ်ရဲ့ နောက်ကွယ်က Logic ကို သိရှိခဲ့ပါတယ်။
  - AI က တစ်ကြိမ်တည်းနဲ့ ဇွတ်အတင်း မပြောင်းဘဲ **Learning Rate** ကို သုံးပြီး ဖြည်းဖြည်းချင်း မျှခြေဖြစ်အောင် Weights တွေကို လိုက်ညှိသွားတာ ဖြစ်ပါတယ်။
